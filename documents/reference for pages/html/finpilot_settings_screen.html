Design decisions

Profile hero with stats

Transaction count, active goals, and a usage streak sit below the avatar. The streak (62 days) is a subtle engagement mechanic — users who see it are more likely to open the app daily.

Month start date setting

Critical for accuracy — someone paid on the 25th wants their budget cycle to start on the 25th, not the 1st. Without this, every report and insight is misaligned.

Category management in settings

Users can add, rename, or hide categories here — keeping the add transaction screen clean while still giving full control to power users.

Violet toggles = on

The toggle uses violet when active — same color as all interactive elements. Off state is a neutral gray. Promotions is the only toggle set to off by default.

AI weekly digest toggle

A Sunday summary push notification is one of the highest-value features for retention — it re-engages users who haven't opened the app mid-week.

Danger zone — no red background

Destructive actions use red text and icons but the same dark card background — no alarming red background that would feel aggressive. The label "Danger zone" sets the context.

Sign out is muted, not red

Sign out is a routine action, not a destructive one — it gets a neutral gray button so it doesn't compete visually with the actual danger zone actions above it.
