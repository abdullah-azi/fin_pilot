Design decisions

Logo placement

Top-left, small — auth screens shouldn't feel like marketing pages. The logo grounds the user without dominating.

Tab switcher

Login ↔ Sign up as a pill toggle keeps both flows on one screen — no separate route needed, less navigation friction.

Violet = interactive

The CTA button, active input border, eye icon, and links are all violet. It's the only color that means "tap this."

Green = validated

Checkmark on confirmed fields. Same green as income on the home screen — consistent signal language.

Amber = warning

Password strength bars use green → amber → red. Subtle but teaches the user about security without lecturing.

No illustration

Dark minimal auth screens don't need hero art. Whitespace and hierarchy do the work — keeps it feeling fast.
